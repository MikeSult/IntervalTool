Interval Tool

This page can be used in two different ways.

!. Enter two letter names and choose a direction (up or down), then click 'calcuation Interval' button (the interval name will be calculated).

2. Enter the first note, an interval name and direction then click 'calculate interval' button (the second note will be calculated).

Use Capital letters for note letter names, use # (hashtag) for sharps, and use b (lowercase B) for flats (i.e. A, Bb, C#, etc.). Try to avoid using double sharps (x) and double flats (bb) if possible. NOTE: some letter combinations are overly confusing and should not be used (i.e. Cx to Fb). Instead it should be rewritten as (D to Fb) or (Cx to E). But even better, for these tones, the most normal choice of letter names is (D to E). Simple is good, no need to over complicate things if you don't have to.

For interval names use the following prefix abbreviations.

  • P = Perfect (i.e. P5)
  • ma = major (ma3)
  • mi = minor (i.e. mi3)
  • dim = diminished (i.e. dim7)
  • aug = augmented (i.e. aug2)
